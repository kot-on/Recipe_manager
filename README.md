# 🍲 Recipe Manager
Данная программа представляет собой приложение написанное на языке Python для записи и управлением рецептов. Данное приложение содержит в себе базу данных (БД). База данных реализована через SQLite.
В наше время очень много информации и рецептов, но нет инструмента для их структуризации. Данное приложение поможет управлять и структурировать рецепты, которые человек хочет приготовить.

---

### 👨‍💼 Разделение ролей
Для этого проекта была собрана команда из 3 человек и у каждого человека есть своя роль. 
- Kronkon - работа с UI и логика входа в программу, обращение к бд, сравнение хэшей пароля.
- Godvar111 - работа с поднятием базы данных и подгрузкой рецептов из базы данных, а также будет выполнена задача по шифрованию базы данных. Это нужно для исключения возможности к просмотру содержимого БД через сторонние источники.
- ?? - работа с логами и резервным копированием БД, также ответственнен за документацию. Также будет выполнено шифрование канала связи с БД для защиты от SQL инъекций.

### 🧰 Инструменты 

Главный инструмент разработки - Python.

Библиотеки Python - Tkinter, Hashlib, SQLcipher, os, shutil.

Для реализации базы данных был выбран SQLite.

### 📋 Запуск приложения

Вам нужно установить [python](https://www.python.org/downloads/) на вашу систему. Затем вам нужно установить библиотеки из файла requirements.txt. Это можно сделать командой pip install requirements.txt и запустите файл Main.py готово!
